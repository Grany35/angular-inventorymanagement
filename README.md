
# Envanter Takip Sistemi FrontEnd

Envanterinize ürün ekleyebilir, konumlarını görebilir ve satışını yapabilirsiniz.


## Özellikler

- Ürün ekleme
- Satış sonrası stok azaltma
- Müşteri ekleme
- Ürünlerin konumlarını takip etme
